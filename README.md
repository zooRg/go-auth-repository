# go-auth-repository

use `go run cmd/main.go`

##### user method
`POST /auth/registration`

`POST /auth/login            `

##### list method
`POST /lists/                `

`GET /lists/                `

`GET /lists/:id             `

`PUT /lists/:id             `

`DELETE /lists/:id             `

##### api method
`POST /api/:id/items/        `

`GET /api/:id/items/        `

`GET /api/:id/items/:item_id`

`PUT /api/:id/items/:item_id`

`DELETE /api/:id/items/:item_id`
